
  # Mobile Dashboard for Nail Salon

  This is a code bundle for Mobile Dashboard for Nail Salon. The original project is available at https://www.figma.com/design/tmJplPbfTFoWaNNgnAobkY/Mobile-Dashboard-for-Nail-Salon.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

## Backend configuration

Set `VITE_API_BASE_URL` to target a backend URL. If omitted, the app defaults to:

`http://0.0.0.0:8200`
